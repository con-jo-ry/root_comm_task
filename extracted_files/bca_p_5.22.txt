punar evaṃ karaṇīyam ity atrārthe dṛḍham abhiniveśaṃ darśayann āha.

civarapiṇḍapātādayo naśyantu. vilayaṃ yāntu mama kāmaṃ yatheṣṭam. satkāro gauraveṇāsanadānapādavandanādipūjā kāyo jīvitaṃ ca sarvam etan naśyatu. anyad api yatkiṃcit sukhasaumanasyanimittaṃ tad api naśyatu. kuśalaṃ punar mama cittaṃ mā kasmiṃścid api kāle naṅkṣīd iti.
