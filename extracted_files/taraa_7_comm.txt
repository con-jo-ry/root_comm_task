saṃpūryetyādi. dānādipāramitā eva guṇā, guṇyante 'bhyasyanta iti kṛtvā. tān saṃpūrya paripūrṇān kṛtvā, yat saṃbuddhānāṃ kṛtyaṃ sakalam avaśyakartavyaṃ kṛtsnaṃ tad api kṛtvā, bhūtakoṭeḥ śūnyatālakṣaṇāyāś cittacaittanirodhātmikāyā yat sākṣāt karaṇaṃ tad api sādhyam astīti pāramitānayasthā evaṃ bruvate caturthaṃ sādhyam iti.
